# Changelog

## [1.0.0] - 2026-05-14
### Added
- 元素识别模式：Click 捕获 CSS Selector + 属性 + 上下文
- 仿写模式：Cmd+C 生成完整设计规格书（视觉/动画/结构）
- 多选模式：Shift+Click 批量选中，Enter 批量复制
- iframe 支持：all_frames 注入 + 跨 frame 广播
- 三种输出格式：YAML / JSON / Markdown
- 智能选择器引擎：id > testid > aria-label > class > nth-path
- @keyframes 提取 + 动画库检测
- Cmd+Shift+E 快捷键支持
- Popup 界面：模式切换 / 格式选择 / 状态指示
- AI 协作治理体系落地（AGENTS.md / .rules / WORKFLOWS）

### Changed
- 项目名称从 ElementSnap 改为 EasyTalkToAI

## Detailed Description (English)

### What It Does

EasyTalk AI is a Chrome extension that converts any webpage UI element into a structured, AI-readable component spec. Click an element — the extension captures its CSS selector, visual properties, layout context, spacing, animations, and more. Paste the output into ChatGPT, Claude, Cursor, or Codex.

### Who It Helps

- **Designers** — Hand off UI specs to AI-powered development tools
- **Frontend developers** — Give Claude Code, Cursor, or Codex exact element context
- **AI coding users** — Stop describing UI in words. Click and paste.

### Main Features

**Click Mode** — Click any element. Copy CSS selector, attributes, layout context, and sibling info. Perfect for: "Make this button more compact."

**Cmd+C Mode** — Press Cmd+C to generate a full design specification: visual properties, layout, interaction states, CSS animations, and a cleaned HTML skeleton. Perfect for: "Recreate this card in React."

**Shift+Click Mode** — Select multiple elements, then press Enter to batch copy. Perfect for: "Refactor these three components."

**Output Formats** — YAML (default), JSON, or Markdown.

**iframe Support** — Works inside embedded iframes.

**Smart Selector Engine** — Automatically picks the best selector: id → data-testid → aria-label → class → nth-path.

**Animation Detection** — Detects framer-motion, GSAP, Lottie, AOS, and CSS @keyframes.

### Example Use Cases

- "Convert this button's style into Tailwind classes"
- "Recreate this hero section in React"
- "Explain the layout of this pricing table"
- "Generate a design-system-friendly component spec"

## 详细描述（中文）

### 功能简介

EasyTalk AI 是一款 Chrome 扩展，可以把网页上的任意 UI 元素转换成 AI 能理解的结构化组件描述。点击元素 → 复制描述 → 粘贴到 ChatGPT、Claude、Cursor 或 Codex。

### 三种模式

- **点击模式**：点击任意元素，复制 CSS 选择器、属性、布局上下文
- **Cmd+C 模式**：生成完整设计规格（视觉、布局、动画、结构骨架）
- **Shift+Click 多选模式**：批量选中多个元素，一次性复制

### 隐私说明

- 纯本地运行，不调用外部 API
- 不上传网页内容到任何服务器
- 无分析、无追踪、无数据收集

## Permissions Explanation

| Permission | Reason |
|------------|--------|
| `activeTab` | Access the current webpage to read element properties (only when activated) |
| `scripting` | Inject the inspector script into web pages |
| `all_frames` | Support iframes by injecting into all frames |
| `clipboardWrite` | Copy the generated spec to clipboard automatically |
| `storage` | Save your preferred output format (YAML/JSON/Markdown) |
| `webNavigation` | Detect iframes on the page |

**No network permissions are requested.** This extension never sends data to a remote server.

# Privacy Policy — EasyTalk AI

*Last updated: 2026-05-15*

## Summary

EasyTalk AI is a Chrome Extension that inspects webpage elements and generates structured descriptions for use with AI tools.

**This extension does not collect, store, transmit, or share any personal data.**

---

## What Data Is Collected

**None.**

This extension does not collect any data.

---

## What Data Leaves Your Browser

**No data leaves your browser.**

- The extension inspects webpage elements **locally** using Chrome's DOM API
- Generated descriptions are copied to your **local clipboard**
- You — the user — decide whether and where to paste the output
- This extension does **not** call any external API, server, or endpoint
- This extension does **not** upload webpage content anywhere
- This extension does **not** send network requests of any kind

---

## What Data Is Stored

The only data stored is:

- **Output format preference** (YAML / JSON / Markdown) — saved in `chrome.storage.local`

This is a simple user preference stored entirely in your browser. It never leaves your device.

---

## Permissions

See [STORE_LISTING.md](./STORE_LISTING.md#permissions-explanation) for a plain-language explanation of Chrome permissions.

All requested permissions are used solely for local element inspection and clipboard access.

---

## Third Parties

This extension:
- Has no backend server
- Uses no third-party analytics
- Uses no tracking scripts
- Uses no cookies
- Contains no advertisements

---

## Changes to This Policy

If this policy changes, the "Last updated" date at the top will be updated. For significant changes, a notice will be added in the extension repository.
